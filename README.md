# AstrBot 插件：网易云音乐 (astrbot_plugin_netease_music)

## 插件简介

**astrbot_plugin_netease_music** 是一款为 AstrBot 打造的网易云音乐集成插件。该插件深度整合了网易云音乐的各项功能，允许用户在机器人端登录网易云账号，并通过指令或大语言模型的函数调用（Function Calling）功能，无缝获取歌曲详情、动态歌词以及热门评论。此外，插件还特别适配了 QQ 平台，支持在聊天中直接发送精美的结构化音乐卡片，极大丰富了群组与私聊的音乐互动体验。

## 功能说明

*   **账号状态管理**：支持用户安全地设置和验证网易云音乐的登录状态，确保能够获取需要登录权限的歌曲信息或评论数据。
*   **大模型函数工具 (Function Calling)**：为大语言模型提供专属工具，使 LLM 能够自主调用网易云音乐 API，智能回答用户关于“某首歌的歌词是什么”、“这首歌有哪些精彩评论”等自然语言提问。
*   **富文本音乐卡片**：针对 QQ 等支持富媒体消息的平台，支持一键发送网易云音乐的专属播放卡片，点击即可在对应客户端内收听。
*   **多维度信息查询**：提供精准的歌曲检索能力，支持通过歌曲名或 ID 快速提取包含歌词、专辑、热评在内的综合数据。

## 插件流程

本插件的内部运行逻辑分为三个主要模块：登录鉴权、大模型函数调用、以及消息链构建。

1.  **登录鉴权与持久化流程**：
    *   当用户触发 `/ncm_login` 指令时，插件会接收用户提供的凭据（如 Cookie 字符串）。
    *   插件内部调用 `pyncm` 库的鉴权接口向网易云音乐服务器验证凭据的有效性。
    *   验证通过后，插件会将登录状态和必要的凭证以 JSON 格式安全地序列化到 AstrBot 的本地 `data/` 目录下。**该过程严格遵循隐私保护原则，凭据仅保留在本地用于合法请求，绝不进行任何未经授权的收集或向第三方服务器上传。**
2.  **大模型函数调用 (Function Calling) 流程**：
    *   插件通过 `@filter.llm_tool` 装饰器向系统注册了获取歌曲信息的函数工具。
    *   当用户在聊天中发送如“帮我查一下网易云音乐《晴天》的热门评论”时，大模型会分析意图，并触发 `ncm_info` 对应的工具函数。
    *   插件接收到 LLM 传递的参数（歌曲名），调用 `pyncm` 接口获取目标歌曲的 JSON 数据（包含 ID、歌词、热评）。
    *   插件将获取到的数据清洗、格式化后回传给大模型，最终由大模型组织自然语言回复发送给用户。
3.  **音乐卡片发送流程**：
    *   当用户执行 `/ncm_card` 指令时，插件首先解析传入的歌曲参数，并在网易云音乐库中检索出唯一匹配的歌曲 ID。
    *   插件识别当前触发事件的平台适配器（如 `aiocqhttp` 或 `Napcat`）。
    *   若为 QQ 平台，插件会利用 AstrBot 的 `MessageChain` 机制，构建包含特定 XML/JSON 结构的 `Comp.Music`（音乐组件）或自定义 Node 消息段。
    *   最后，通过 `yield event.chain_result()` 将构建好的音乐卡片下发至目标会话。

## 使用方法

### 1. 账号登录与验证
使用 `/ncm_login` 指令来设置你的网易云音乐登录状态。
*   **指令语法**：`/ncm_login <登录凭证>`
*   **使用示例**：`/ncm_login MUSIC_U=xxxxxx;...`
*   **说明**：成功后机器人会回复当前登录的用户昵称及状态。

### 2. 查询歌曲信息
通过 `/ncm_info` 指令直接查询歌曲的详细信息。
*   **指令语法**：`/ncm_info <歌曲名称或ID>`
*   **使用示例**：`/ncm_info 岁月神偷`
*   **说明**：机器人会返回该歌曲的完整歌词以及当前排名前列的热门评论。

### 3. 发送音乐卡片
通过 `/ncm_card` 指令在聊天中发送可交互的音乐卡片。
*   **指令语法**：`/ncm_card <歌曲名称或ID>`
*   **使用示例**：`/ncm_card 12345678` 或 `/ncm_card 稻香`
*   **说明**：机器人将发送一张包含歌曲封面、歌名和歌手的网易云音乐卡片（仅限支持该消息类型的平台，如 QQ）。

### 4. 结合大语言模型自然对话
由于插件深度集成了 Function Calling，你无需刻意记忆指令，只需像和真人聊天一样对机器人说：
*   *“帮我找一下网易云里《十年》的歌词。”*
*   *“陈奕迅的《孤勇者》在网易云有什么热门评论吗？”*
*   *“给我发一张《起风了》的网易云音乐卡片。”*（需大模型支持工具组合调用）
